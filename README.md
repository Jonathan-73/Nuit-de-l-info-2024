## Utilisation du code

Notre site utilise **Vite** et **Tailwind CSS**. Pour générer le site,
il vous faudra `Node.JS`.

Clonez le répertoire et exécutez :
```sh
npm run build
```

Les fichiers produits seront dans le dossier `dist`.
